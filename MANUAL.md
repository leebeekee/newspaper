# 신문사설 관리자 시스템 사용 설명서

## 1. 개요
이 시스템은 브라우저를 통해 신문사설 데이터를 입력, 검색, 저장하는 도구입니다. 데이터는 `editor.xml` 파일에 **누적 저장**됩니다.

## 2. 저장 방식의 이해 (중요!)

이 웹사이트는 **정적 웹사이트(HTML/CSS/JS)**입니다. 따라서 실행 환경에 따라 저장 방식이 달라집니다.

### A. 로컬 컴퓨터 (서버 실행 시) - **권장**
`node server.js`를 실행하고 `localhost:3000`으로 접속하면, **자동 저장**이 가능합니다.
- **저장 버튼**: 클릭 시 자동으로 `editor.xml` 파일이 갱신됩니다.

### B. 깃허브(GitHub Pages) 호스팅 시
깃허브 페이지는 **서버 기능이 없는 단순히 파일만 보여주는 공간**입니다. 따라서 웹사이트에서 파일을 직접 수정(저장)하는 것은 **불가능**합니다.
- **저장 버튼**: 클릭 시 [자동 저장]은 실패하고, 대신 수정된 `editor.xml` 파일이 **다운로드**됩니다.
- **데이터 갱신 방법**:
    1. 관리자 페이지에서 수정 후 '저장'을 눌러 `editor.xml`을 다운로드 받습니다.
    2. 다운로드 받은 파일을 깃허브 저장소(Repository)에 **직접 올려서(Upload/Push) 덮어씌워야** 합니다.
    3. 잠시 후 사이트를 새로고침하면 업데이트가 반영됩니다.

### C. 로컬 컴퓨터 (서버 미실행 시)
서버 없이 `file://.../index.html`로 파일을 열면 보안 문제로 데이터 로딩 및 저장이 차단될 수 있습니다.

---

## 3. 로컬 서버 실행 방법 (자동 저장을 위해)
1. 터미널(CMD/PowerShell)을 엽니다.
2. 프로젝트 폴더(`c:\Users\User\glocery`)로 이동합니다.
3. 다음 명령어를 입력합니다:
   ```bash
   node server.js
   ```
4. 브라우저에서 `http://localhost:3000/` 으로 접속합니다.
